# EasyColorsExtension
Extensão de palheta de cores para Bootstrap 5

Link para consultar as cores disponíveis na versão 1.0: https://getbootstrap.com/docs/5.2/customize/color/

COMO USAR 

[bg/text]-[cor]-[variação(opcional)]


Exemplos:

bg-blue
bg-pink-500
text-red
text-orange-100
